Descripción detallada

T.D.A. StackMax.

Una instancia s del tipo de datos abstracto StackMax es un objeto que representa una pila de Element. Está compuesto por una estructura de datos (StackType) v, que puede ser una de tres: un vector dinámico, una lista ó una cola. Este dato constituye la representación interna de una pila de Element. Una StackMax de longitud k la denotamos:

<(num_1,max_1), (num_2, max_2),...,(num_k, max_k)>
