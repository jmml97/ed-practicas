Descripción detallada

template<class T>
class DVector< T >

T.D.A. DVector.

Una instancia v del tipo de dato abstracto DVector (vector) sobre un dominio T es una sucesión finita de elementos del mismo. En un vector, la inserción se puede realizar en cualquier posición. De igual forma, podemos obtener el elemento de la posición que queramos. Un vector de n elementos lo representamos:

<a1,a2,...,an>

Los vectores tienen un cierto tamaño, que puede ampliarse o reducirse en cualquier momento. Si n = 0, diremos que el vector está vacío.
